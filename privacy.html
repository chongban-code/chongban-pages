宠伴隐私政策

更新日期：2026年6月11日
生效日期：App Store 首次发布日起生效

宠伴是一款私密记录和宠物陪伴工具。我们把“少收集、先说明、用户可控”作为首版隐私原则。

我们保存什么

  • 宠物资料：名称、类型、头像、本地创建时间。
  • 聊天内容：文字消息、图片消息、语音消息、本地文件路径、会话时间。
  • 设置项：隐私锁、宠物语、统计开关、宠伴权益、导出次数。

这些内容默认保存在你的设备本地，不会因为使用基础功能而上传到我们的服务器。

系统权限

  • Face ID / Touch ID / 设备密码：用于启动隐私锁，验证后进入 App。
  • 相册：用于选择宠物头像或聊天图片。
  • 麦克风：用于录制用户主动发送的语音消息，未使用语音功能时不会录音。

统计分析

如果你同意启用匿名统计，我们会通过友盟等第三方统计服务了解功能使用情况，例如启动、创建宠物、发送消息、导出备份、购买流程状态。

我们不会把聊天正文、宠物名称、图片、语音、备份内容或本地文件路径发送给统计服务。

你可以在 App 内设置页关闭匿名统计。关闭后，后续使用行为不再主动发送到统计服务。

付费与购买

宠伴内购由 Apple App Store 处理。我们通过 StoreKit 获取购买状态，用于解锁宠伴 Pro 功能。付款信息由 Apple 处理，我们不会获取你的银行卡或支付凭据。

如果你购买自动续订订阅，订阅、续费、取消和退款由 Apple 按 App Store 规则处理。你可以在 Apple ID 的订阅管理中取消自动续订。

数据导出

你可以在设置页导出 JSON 备份。1.0 的 JSON 备份包含宠物、对话、文字消息、图片和语音消息的本机文件引用，不会把图片或语音文件本身打包进去。备份文件由你自行保存或分享，请妥善保管。
